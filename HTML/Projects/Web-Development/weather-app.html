Weather App Project

A Weather App that provides real-time weather forecasts

This was probably my first and most favourite project because it literally showed me a world I did not know existed. This project is from my SheCodes Web Development certification.

It introduced me to things like APIs and hosting. The project offers real-time and accurate weather forecasts and it is hosted on Netify.

Technologies used:

  1. HTML
  2. CSS
  3. JavaScript
  4. OpenWeatherMap API
  5. Netlify (for hosting)

Functionality:

The app allows users to search for a city and view the current weather conditions, including temperature, humidity, wind speed, and a brief description of the weather. It also displays an icon representing the current weather condition.
